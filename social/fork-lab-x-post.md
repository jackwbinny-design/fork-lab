Main post:

```text
I kept forking interesting GitHub repos and telling myself I’d “look at them later.”

Later never came.

So I built Fork Lab:
an open-source dashboard that turns a fork backlog into a weekly learning queue.

It helps me:
- group related repos into clusters
- pick what to explore this week
- keep / deepen / drop projects explicitly

If your GitHub account is full of promising forks but no clear next step, this might be useful.

Repo: https://github.com/jackwbinny-design/fork-lab
```
